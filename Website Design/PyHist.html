Python's History

Python first started with a general-purpose programming
language called ABC. ABC was created in the Netherlands,
and it’s greatest accomplishment was to influence Python’s
design.

The idea of Python was created in the late 1980’s when Guido van
Rossum was working on an operating system called Amoeba. He also
worked on ABC, and said in an interview that he became frustrated
with the language because it had many problems. Rossum wanted to
create a language that contained most of ABC’s properties but without
all of it’s problems. So he worked on a scripting language with all
the parts of ABC he liked and a few of his own parts that have never
been seen in a programming language before.

In December, 1989, Rossum chose the name ‘Python’ for his language.
He chose this name because he was a big fan of Monty Python’s Flying
Circus, a British comedy series. Python was later released as a programming
language in 1991 as version 0.9.0. Then, in 1994, Python 1.0 was released.
Soon after that, Python 2.0 was released in October 2000, and for another
8 years, it flourished. Finally Python 3, or Py3k was released adding many
new features. Overall, Python is a very successful language, and how
easy it is to learn it makes it perfect for new programmers.
